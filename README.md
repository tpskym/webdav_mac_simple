# WebDAV Client (macOS)

Нативное приложение для macOS: подключение к WebDAV по **HTTPS**, просмотр каталогов и сохранение файлов на диск.

## Требования

- macOS 13+
- Xcode 15+ или Swift 5.9+ (Command Line Tools)

## Сборка и запуск

### Из терминала (рекомендуется)

**При каждой сборке** запускайте скрипт — он собирает проект, копирует бинарник в бандл и подкладывает иконку (иначе в `WebDAVClient.app` может не быть папки Resources и иконки):

```bash
cd webdav
./scripts/build-and-bundle.sh
open WebDAVClient.app
```

Только сборка без упаковки в .app:

```bash
swift build
.build/debug/WebDAVClient
```

### Из Xcode

1. Открыть папку `webdav` в Xcode: **File → Open** → выбрать папку проекта.
2. Выбрать схему **WebDAVClient**, запустить (⌘R).

## Использование

1. **Сервер** — укажите полный HTTPS URL WebDAV (например `https://cloud.example.com/remote.php/webdav`).
2. **Логин и пароль** — учётные данные для Basic-аутентификации.
3. **Обновить список** — запрос PROPFIND и отображение содержимого текущей папки.
4. Двойной клик по папке — переход в каталог; **Вверх** — на уровень выше.
5. **Папка для сохранения** — выбрать локальную директорию для скачивания.
6. Выделить файлы в списке и нажать **Скачать выбранные** — файлы сохраняются в выбранную папку.

Поддерживаются только **HTTPS**; логин и пароль передаются по Basic Auth.

## Структура проекта

- `WebDAVClient/`
  - `WebDAVClientApp.swift` — точка входа SwiftUI.
  - `ContentView.swift` — экран: подключение, список, скачивание.
  - `WebDAVService.swift` — протокол WebDAV: PROPFIND (список), GET (скачивание), разбор XML.
  - `Models.swift` — модели `WebDAVResource`, `WebDAVCredentials`.

## Тесты

Автоматические тесты не добавлены: приложение завязано на GUI (SwiftUI) и сетевые вызовы (WebDAV). Для покрытия потребовались бы моки `URLSession` и отдельный тестовый WebDAV-сервер или стабы ответов PROPFIND/GET.
